# Feature Showcase Flow

Present features in this order for maximum narrative impact.

## 1. Instant SOS (hero)
- Gold 3D SOS button, countdown, cancel window
- **Demo:** Tap SOS or Run Hackathon Demo

## 2. Multi-trigger safety
- Volume 3×, shake, calculator disguise (`911`)
- AI motion: fall, crash, panic, inactivity
- **Demo:** Press `V` three times; calculator secret code

## 3. Live tracking & contacts
- GPS on map placeholder, event ID, contact blast
- **Demo:** Tracking screen after SOS

## 4. Offline resilience (differentiator)
- Offline banner, sync queue, SMS fallback, mesh relay
- **Demo:** Toggle offline in DevTools; show queue + sync

## 5. Command center (B2G)
- WebSocket live feed, incident panel, assign responder
- **Demo:** Launch live demo on dashboard

## 6. Danger Zone AI (innovation)
- Heatmap, hotspots, risk score, route safety
- **Demo:** Risk AI view + heatmap toggle

## 7. Analytics & ops
- Active incidents, 24h volume, response time
- **Demo:** Analytics panel bottom-left
