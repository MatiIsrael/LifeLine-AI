## Quick start (judges)

| Step | Action | URL |
|------|--------|-----|
| 1 | Overview | http://localhost:4000 (home) |
| 2 | Victim app demo | http://localhost:3000 → **Run Hackathon Demo** |
| 3 | Command center | http://localhost:5173 → key `lifeline-dashboard-dev` → **Launch Live Demo** |
| 4 | Pitch deck | Open `hackathon/pitch-deck.html` in browser |

```powershell
cd backend && npm run dev
cd dashboard && npm run dev
npx serve preview -l 3000
npx serve home -l 4000
```
